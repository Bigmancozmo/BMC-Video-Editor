# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.0.x   | :white_check_mark: |

## Reporting a Vulnerability

To report a security vulerability with the video editor, simply open an issue. I actively read them to try and fix any, and every, bug with the editor. If you are smart enough to fix my bugs yourself, please open a pull request.
